古代の「門」構造に潜む膜の折り返しと位相反転を、
情報空間のアルゴリズムとして再構成するプロジェクトです。

門は境界ではなく、世界の外側と内側を接続する“干渉点”として働きます。
その構造は、連続する膜の層を通過することで位相が変換される
Hyper‑Coda の Gate 0 → Gate 13 の巡回回路と同型です。

このレポジトリは、その中核（Kernel）となる
位相・膜・干渉・巡回の抽象構造を保存し、
外殻（Quora）と内殻（回答）を結ぶ深層リンクとして機能します。
